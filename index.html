Hive Token Generator

This is an unofficial 3rd party site for generating Hive Authentication Tokens designed for use with Smart Things Handler.

Disclamer

I offer no gurantees on the functionality of this service. I am not affiliated with Hive in any way. This service will take your hive credentials and product JSON tokens for use with Smart things. The tokens pass through this site, nothing is stored. You SHOULD be skeptical about using services like this. Take a moment to understand this, I can't tell you to trust me / my server to temporarly handle your credentials, you need to make that decision.

You can find more information about why this was designed here: https://community.smartthings.com/t/release-hive-connect-v3-2c-british-gas-hive/103473 I have taken the example provided and rolled it into a service which is easier to use for non-programmers.
